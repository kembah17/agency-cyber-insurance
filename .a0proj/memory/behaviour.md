*   **Design:** Avoid simplistic flat designs; explore 3D metallic/glass styles.
*   **Logos/Icons:** Use brand-identity skill's Tier 2/3 pipeline (Aura-Enhanced Prompt Engineering, refinement loops, vision QA).

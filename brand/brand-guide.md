### Logo Usage Rules

|------|--------------|
| **Minimum Size** | 32px height (digital), 10mm (print) |
| **Clear Space** | Minimum 1x logo height on all sides |
| **Primary Background** | White or light neutral |
| **Dark Background** | Use reversed (white) version |
| **Colored Background** | Use monochrome version |

### Logo Don'ts

- Do not stretch, skew, or rotate the logo
- Do not change the logo colors outside brand palette
- Do not add effects (shadows, gradients, outlines)
- Do not place on busy or low-contrast backgrounds
- Do not rearrange logo elements
- Do not use the logo smaller than minimum size

## 5. Brand Voice & Tone

**Overall Tone**: Confident, knowledgeable, and approachable

### Do

- Use clear, direct language
- Back claims with evidence
- Address the reader respectfully
- Use industry terminology appropriately

### Don't

- Use slang or overly casual language
- Make unsubstantiated claims
- Be condescending or jargon-heavy
- Use excessive exclamation marks

### Example Phrases

> We deliver results you can measure.

> Our expertise ensures your success.

> Let us show you a better way.

### Tone Spectrum

| Context | Tone Adjustment |
|---------|----------------|
| Marketing/Ads | More energetic and persuasive |
| Website Copy | Balanced, clear, and engaging |
| Support/Help | Empathetic, patient, solution-focused |
| Social Media | Conversational, timely, personality-forward |
| Legal/Formal | Precise, respectful, straightforward |
| Internal | Collaborative, transparent, encouraging |

## 6. Visual Style

### Photography Style

| Attribute | Guideline |
|-----------|-----------|
| **Lighting** | Natural, well-lit, avoid harsh shadows |
| **Subjects** | Authentic people, real environments |
| **Color Treatment** | Consistent with brand palette, slight warm/cool tint |
| **Composition** | Clean, uncluttered, focus on subject |
| **Avoid** | Stock photo cliches, overly staged scenes |

### Iconography

- **Style**: Outlined, 2px stroke, rounded caps
- **Size**: 24px base grid
- **Color**: Use primary or neutral-600 on light backgrounds
- **Consistency**: Same weight and style across all icons

## 9. Implementation Guide

### CSS Integration

Import the generated CSS file for instant brand colors:

```html
<link rel="stylesheet" href="palette.css">
```

### Tailwind CSS Integration

Merge `tailwind-colors.json` into your `tailwind.config.js`:

```javascript
const brandColors = require('./tailwind-colors.json');

module.exports = {
  theme: {
    extend: {
      colors: brandColors.colors,
    },
  },
};
```

### Logo Generation

Use the `openrouter-image-gen` skill with the prompts in `logo-prompts.json`:

```bash
python /a0/usr/skills/openrouter-image-gen/generate_image.py \
  --prompt "$(cat logo-prompts.json | python -c "import sys,json; print(json.load(sys.stdin)[0][\"prompt\"])" )" \
  --api-key YOUR_KEY \
  --output ./logos/concept-1.png
```
